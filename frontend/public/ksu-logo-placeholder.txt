KSU Logo Placeholder

To add the Kennesaw State University logo:

1. Save your KSU logo file as: frontend/public/ksu-logo.png
   (or ksu-logo.svg, ksu-logo.jpg - any format works)

2. Update src/components/Header.js:
   Replace the logo-placeholder div with:
   
   <img 
     src="/ksu-logo.png" 
     alt="Kennesaw State University Logo" 
     className="ksu-logo"
   />

3. Add CSS in Header.css:
   .ksu-logo {
     width: 50px;
     height: 50px;
     object-fit: contain;
   }

The logo will automatically appear in the header!
